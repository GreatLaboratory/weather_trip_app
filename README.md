# backend

## Running & Building webserver
```
$ cd backend
$ pip install flask, flask_cors, tensorflow
$ python app.py
```

# frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
$ cd frontend
$ npm run serve
```

### Compiles and minifies for production
```
$ cd frontend
$ npm run build
```

### Run your tests
```
$ cd frontend
$ npm run test
```

### Lints and fixes files
```
$ cd frontend
$ npm run lint
```
